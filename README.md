## 딥러닝(Deep Learning) 이란?
`딥 러닝`은 **머신 러닝의 한 방법**으로, 학습 과정 동안 인공 신경망으로서 예시 데이터에서 얻은 `일반적인 규칙을 독립적으로 구축(훈련)`합니다. 

특히 머신 비전 분야에서 신경망은 일반적으로 데이터와 예제 데이터에 대한 사전 정의된 결과와 같은 `지도 학습`을 통해 학습됩니다.
